# Case Sensitive

## Definición

cuando hablo de Case Sensitive me refiero a que el lenguaje diferencia entre mayúsculas y minúsculas.

## Ejemplo

las siguientes cadenas serían consideradas diferentes con case sensitive:

- "Hola" y "hola"
- "Gato" y "gato"
- "OpenAI" y "openAI"
